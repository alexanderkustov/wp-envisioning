# Envisioning 0.1
__A Wordpress starter theme based on Bones and Bootstrap 3__

![Envisioning](screenshot.png)


Features:
* LESS
* Bootstrap 3
* Bones
* sidebar and footer widgets
* Very clean out-of-the-box experience
* pagination
* breadcrumbs
* Font Awesome
* Redux Options Framework
* Easy to add metaboxes
